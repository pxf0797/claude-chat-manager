# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-15

### 新增
- **核心功能**: 完整的 Claude Code 聊天记录管理方案
- **查看工具**: `view-chats.sh` - 快速查看聊天记录
- **交互浏览器**: `chat-explorer.sh` - 交互式聊天浏览器
- **导出工具**: `export-to-obsidian.sh` - 基础导出到 Obsidian
- **增强导出**: `export-enhanced.sh` - 支持双链笔记的增强导出
- **自动监控**: `chat-monitor.sh` - 监控新对话并自动导出
- **安装工具**: `install-chat-tools.sh` - 一键安装脚本
- **测试工具**: `test-chat-tools.sh` - 环境测试脚本

### 功能特性
- 支持查看原始聊天记录（JSONL 格式）
- 支持导出到 Obsidian 等双链笔记工具
- 自动提取话题标签和元数据
- 按日期、项目、话题自动分类
- 支持 Dataview 查询和统计
- 可配置的自动监控服务
- 完整的日志记录和错误处理

### 文档
- `README.md` - 项目说明和快速开始
- `Claude-Code-Chat-Management-Scheme.md` - 完整方案文档
- `Quick-Start-Guide.md` - 快速开始指南
- `CONTRIBUTING.md` - 贡献指南
- `GITHUB_SETUP.md` - GitHub 仓库设置指南
- `LICENSE` - MIT 许可证

### 依赖
- **jq**: JSON 处理工具（必需）
- **bash**: Shell 环境（版本 4.0+）
- **系统工具**: find, date, awk, sed, grep 等

## 版本规则

### 版本号格式：主版本号.次版本号.修订号

1. **主版本号**：不兼容的 API 修改
2. **次版本号**：向下兼容的功能性新增
3. **修订号**：向下兼容的问题修正

### 发布周期
- **主要版本**：每 6-12 个月发布一次
- **次要版本**：每 1-2 个月发布一次
- **修订版本**：根据需要随时发布
